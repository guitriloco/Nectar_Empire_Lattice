# ⚜️ Estrutura de Nichos (Expansion)

Este diretório contém os ativos de marketing, estratégia e técnicos organizados por nicho, prontos para injeção e expansão do império.

## 📂 Estrutura

- **nectar_divino/**: Ativos da marca core e sistema principal.
- **wealth/**: Ativos do nicho de Riqueza Soberana (Nectar_Wealth).
- **fitness/**: Ativos para o nicho de saúde e corpo ideal.
- **pets/**: Ativos para o nicho de animais de estimação.
- **general_strategy/**: Insights globais e estratégias de escalonamento que transcendem nichos específicos.

## 🚀 Como Usar
Escolha o nicho de expansão, utilize os prompts da `cave_gemini` e os ativos deste diretório para materializar a nova instância do império com fricção zero.
