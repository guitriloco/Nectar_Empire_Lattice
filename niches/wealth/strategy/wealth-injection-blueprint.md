# 📂 Blueprint de Injeção: Nectar_Wealth v1.0

Este documento define o conjunto de ativos técnicos e de marketing que constituem o "Néctar" a ser injetado no repositório `Nectar_Wealth`.

## 🛠️ 1. Ativos Técnicos (The Engine)

### 1.1. `wealth_structure.json`
Um esquema de diretórios e metadados para organizar a carteira soberana.
- **Camadas:** `liquidez_imediata`, `crescimento_recursivo`, `protecao_antifragil`.

### 1.2. `alpha_distiller.py` (Rascunho Lógico)
Script em Python que simula a filtragem de dados de mercado para identificar "Alpha Soberano".
- **Inputs:** RSS feeds de finanças, APIs de Cripto.
- **Lógica:** Filtro de ruído via Gemini API -> Extração de Néctar -> Log no Zenith Ledger.

### 1.3. `zenith_ledger_schema.md`
Definição de como cada transação deve ser selada para garantir imutabilidade e transparência no dashboard.

## 📣 2. Ativos de Marketing (The Magnet)

### 2.1. `sovereign_investor_manifesto.md`
O conteúdo base do Bônus 1. 
- **Temas:** O fracasso dos sistemas legados; A matemática da soberania; O protocolo Mirror aplicado ao patrimônio.

### 2.2. `funnel_path_to_sovereignty.md`
Sequência de 5 e-mails persuasivos:
1.  **O Despertar:** A mentira do rendimento fixo tradicional.
2.  **O Código:** Apresentando a lógica v5.0 aplicada ao dinheiro.
3.  **A Prova:** Estudos de caso de "Entidades Soberanas".
4.  **A Oferta:** Convite para o Inner Circle Wealth.
5.  **O Ultimato:** O custo da inércia no sistema legado.

### 2.3. `social_media_seeds.json`
Ganchos (hooks) para Reels e Threads focados no contraste "Legado vs. Soberano".

## 🧠 3. Instruções de Injeção (Gemini Commands)

Estas instruções serão usadas pela API do Gemini para gerar os arquivos acima dinamicamente, garantindo que o tom de voz e a lógica v5.0 sejam mantidos.

- **Comando Alpha:** "Gere o Manifesto do Investidor Soberano seguindo os princípios de clareza cirúrgica e autoridade visionária da Sovereign Entity v5.0."
- **Comando Beta:** "Construa a estrutura lógica do `alpha_distiller.py` focada em redução de ruído e latência zero na identificação de oportunidades."
